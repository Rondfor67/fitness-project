# Fitness App

Веб-приложение для фитнес-клуба с возможностью записи на групповые и индивидуальные тренировки.

Проект разработан в рамках курсовой работы по дисциплине «Проектирование веб-сайта и обеспечение его бесперебойной работы».

---

## Описание проекта

Данное приложение представляет собой полноценную клиент-серверную систему.

Пользователь может:
- зарегистрироваться и войти в систему
- просматривать доступные тренировки
- записываться на групповые и индивидуальные занятия
- отменять запись (не менее чем за 3 часа до начала)
- просматривать список своих записей

Администратор может:
- добавлять новые тренировки
- редактировать тренировки
- удалять тренировки
- управлять расписанием и доступными слотами

---

## Используемые технологии

Frontend:
- React (Vite)
- JavaScript (ES6+)
- React Router
- CSS

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT (аутентификация)
- bcrypt (хеширование паролей)

Деплой:
- Vercel (frontend)
- Render (backend)

---

## Архитектура проекта

Приложение построено по клиент-серверной архитектуре:

- Frontend отвечает за интерфейс и взаимодействие с пользователем
- Backend обрабатывает запросы и реализует бизнес-логику
- База данных хранит пользователей, тренировки и записи

Связь осуществляется через REST API.

---

## Основной функционал

Аутентификация:
- регистрация пользователя
- вход в систему (JWT)

Групповые тренировки:
- просмотр тренировок
- запись
- ограничение по количеству мест
- отмена записи

Индивидуальные тренировки:
- выбор тренера
- выбор времени
- бронирование
- возврат свободного времени при отмене

Записи:
- просмотр своих записей
- отмена записи с проверкой времени

---

## API маршруты

Аутентификация:
- POST /api/register
- POST /api/login

Групповые тренировки:
- GET /api/trainings
- POST /api/trainings
- POST /api/trainings/:id/join
- PUT /api/trainings/:id
- DELETE /api/trainings/:id

Индивидуальные тренировки:
- GET /api/trainers
- POST /api/trainers
- POST /api/trainers/:id/book
- PUT /api/trainers/:id
- DELETE /api/trainers/:id

Записи:
- GET /api/my-bookings/:userId
- DELETE /api/bookings/:id

## Установка проекта

1. Клонировать репозиторий:

git clone https://github.com/your-username/fitness-project.git

---

2. Backend:

cd server  
npm install  

Создать файл .env:

MONGO_URI=ваша_строка_подключения  
JWT_SECRET=секрет  
PORT=5000  

Запуск:

node server.js

---

3. Frontend:

cd client  
npm install  
npm run dev  

---

## Переменные окружения

Backend:
- MONGO_URI — подключение к MongoDB
- JWT_SECRET — секрет для токенов
- PORT — порт сервера

Frontend:
- VITE_API_URL — ссылка на backend (Render)

---

## Деплой

Frontend размещён на Vercel  
Backend размещён на Render
